如果您使用Git一段时间，那么您应该接触过git log。 众所周知，git log是一个简单的命令，可以帮助我们查看变更或项目历史记录。

即使如此简单，它仍然是一个非常强大的工具，它带有许多参数，可以帮助我们查看项目变更及其结构。 我们将在本文中看到一些最常用的参数。

git log —oneline

此命令可帮助您以更简洁的方式查看提交。 它会将每次提交压缩到一行，并且只有很少的信息，例如较短的提交哈希值和提交消息。
